### Architecture
- tilde-mcp crate: JSON-RPC 2.0 handler with tool dispatch
- McpState stored in AppState, shares db Arc with rest of server
- Tools: notes.{search,read,append}, files.{list,read,search}, trackers.{log,query}
- Auth: Bearer mcp_prod_ tokens validated from mcp_tokens table
- Scope check: per-tool required scope, *:read wildcard, no *:write wildcard
- Rate limiting: in-memory per-token sliding window
- Audit: mcp_audit_log table with auto-prune on retention_days
